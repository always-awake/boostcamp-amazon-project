## 폴더 및 주요 소스 파일 구조
### server (express generator 사용)
```
2_clone_amazon_carousel
└── server
    └── bin
    └── config
        └── db.js
        └── passport.js
    └── public
        └── css
        └── images
        └── js
    └── routes
    └── views (pug 템플릿 엔진 사용)
```
